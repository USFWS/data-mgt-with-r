Title: Habitat covariate
Abstract
Short Abstract:
R data object of formal class 'RasterLayer' containing spatial data

Full Abstract:
R data object of formal class 'RasterLayer' containing simulated spatial data for all possible survey sites within the study area. Spatial data includes x and y coordinates and site-specific values of a hypothetical spatially correlated habitat covariate. Spatial covariate values were generated using primary project script "0_data_simulation.R". This data set is not legitimate spatial data set lacking required attributes such as projection, etc and only serves as a training tool to demonstrate proper data management.
